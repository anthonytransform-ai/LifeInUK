# Life in the UK - Bilingual Mock Test Web App

A serverless, mobile-first web application designed to help Hong Kongers prepare for the "Life in the UK" test. The application provides bilingual (English and Traditional Chinese) mock tests mirroring the official exam format.

---

# Life in the UK - 雙語模擬考試網頁應用程式

一個無伺服器（Serverless）、行動裝置優先（Mobile-first）的網頁應用程式，專為準備「Life in the UK」入籍試的香港人而設。本程式提供英文與繁體中文雙語對照模擬考試，完全模擬官方考試格式。

---

## Features / 功能特點

### English
- **Bilingual Interface**: Seamlessly toggles between bilingual (English + Traditional Chinese) and English-only modes via a global context, with persistence via `localStorage`.
- **Accessible Design**: Engineered for extreme clarity with high-contrast color palettes, a strict mobile-first stacked layout (English over Chinese), and large 48px tap targets, heavily catering to elderly users.
- **Test Engine**: Enforces official rules (24 questions, 45-minute strict countdown timer, 18/24 passing threshold).
- **Results & Review**: Grades the exam, clearly displays right/wrong answers, and injects bilingual explanations directly derived from the official Study Guide for every question.
- **Local Progress Tracking**: The user's dashboard automatically tracks completed tests and scores locally using `localStorage`.
- **Zero Backend**: All mock tests are bundled as static JSON data (`src/data/mock_tests.json`), automatically parsed and generated by an offline AI process using the official study text.

### 中文
- **雙語介面**：透過全域狀態，在雙語對照模式（英文 + 繁體中文）與純英文模式之間無縫切換，並自動保存設定至瀏覽器 `localStorage`。
- **高可讀性設計**：專為年長用戶（如 60 歲以上）進行可讀性優化。採用高對比度色彩、行動裝置優先的上下垂直排列結構（英文在上，中文在下），並提供大於 48px 的大點擊區域，極易操作。
- **考試引擎**：完全遵循官方考試規則（24 道題目、45 分鐘嚴格倒計時、及格分數為 18/24 題即 75%）。
- **結果與回顧**：自動評分，清晰標註正確與錯誤的選項。並針對每道題目提供直接取自官方學習手冊（Study Guide）的雙語答案解析。
- **本地進度追蹤**：儀表板會使用 `localStorage` 自動記錄已完成套卷的歷史得分和通過狀態。
- **無後端架構**：所有模擬試題皆儲存在靜態 JSON 檔案（`src/data/mock_tests.json`）中。

## Tech Stack / 技術棧

- **Framework / 框架**: Next.js (App Router)
- **Language / 語言**: TypeScript
- **Styling / 樣式**: Tailwind CSS v4
- **Testing / 測試**: Vitest + React Testing Library
- **Deployment / 部署**: Static Export to GitHub Pages (`next.config.ts` configured for `output: "export"`)

## Getting Started / 快速開始

### English
First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3001/LifeInUK](http://localhost:3001/LifeInUK) with your browser to see the result. (Due to `basePath` configuration, it will be hosted on `/LifeInUK`).

### 中文
首先，安裝相依套件：
```bash
npm install
```

接著，啟動本地開發伺服器：
```bash
npm run dev
```

在瀏覽器中開啟 [http://localhost:3001/LifeInUK](http://localhost:3001/LifeInUK) 查看成果。

## Save to Homescreen / 儲存至手機主畫面 (PWA)

### English
This application is PWA-compatible. You can save it to your phone's home screen to run it fullscreen like a native mobile app:
- **iPhone / iOS (Safari)**: Tap the **Share** button at the bottom, then select **Add to Home Screen**.
- **Android (Chrome)**: Tap the menu button (three dots) in the top-right corner, then select **Add to Home Screen** or **Install App**.

### 中文
本網頁支援 PWA 規範。在智慧型手機上，您可以將其儲存至桌面，像原生手機 App 一樣全螢幕啟動，不受瀏覽器網址列干擾：
- **iPhone / iOS (Safari)**：點擊底部的「分享」按鈕，然後選擇「加入主畫面」。
- **Android / Chrome**：點擊右上角的選單按鈕，選擇「新增至主畫面」或「安裝應用程式」。

## Deployment / 部署至 GitHub Pages

To deploy to GitHub Pages:
```bash
npm run deploy
```
This triggers the `predeploy` script which builds the static export into `/out`, followed by pushing to the `gh-pages` branch using the `gh-pages` package.
